Chess Game with AI Engines

This project is a fully-featured Chess Game built from scratch in Java, combining OOP design, GUI development, and AI algorithms. Key highlights include:

Object-Oriented Programming (OOP): Abstraction through a Piece abstract class, encapsulation for private attributes like player names, and inheritance/polymorphism to model chess pieces and their moves.

Interactive GUI: Developed using Java Swing for a smooth, user-friendly experience.

Player vs Player Mode: Allows two players to compete, with potential to expand online in the future.

Three AI Difficulty Levels:
- Easy: random legal moves for beginners.
- Medium: prioritizes capturing valuable pieces.
- Hard: implements Minimax with Alpha-Beta pruning, considering material balance, king safety, pawn structure, positional evaluation, and mobility for strategic AI decisions.

This project enhanced my skills in OOP, GUI design, AI strategy, debugging, and teamwork.
